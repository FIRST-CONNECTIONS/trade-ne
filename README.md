# Trade NE

The business alliance for the North East of England — a **First Connections** initiative.

A single-page site (`index.html`) built in the First Connections brand system, championing the region's traders, founders and high streets. Made possible by our delivery partners, including Lumo.

- **Live brand assets** pulled from the First Connections `FC-Logos` repo
- **Deployed** via Netlify (connected to this GitHub repo)
- **Group campaigns:** Newcastle First · Darlington First · NE High Street Heroes
